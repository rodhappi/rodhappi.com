## Desarrollo

Mi entorno de desarrollo es minimalista por diseño. Estas son las herramientas que uso todos los días:

- **Editor:** VS Code con un puñado de extensiones
- **Terminal:** iTerm2 con zsh
- **Control de versiones:** Git + GitHub
- **IA:** Claude para pair programming

No uso muchas extensiones. Las que tengo instaladas las uso *todos los días*. Si una extensión lleva un mes sin activarse, la desinstalo.

## Escritura

Para escribir uso un flujo simple:

1. Abro un archivo Markdown
2. Escribo sin formatear
3. Edito al día siguiente con ojos frescos
4. Publico

No uso Notion, ni Obsidian, ni ningún sistema complejo. Un directorio con archivos `.md` es suficiente.

> Las mejores herramientas son las que desaparecen. No piensas en el martillo mientras clavas el clavo.

## Productividad

Aquí es donde muchos caen en la trampa de la *herramienta perfecta*. Mi sistema es deliberadamente básico:

- **Calendario** para compromisos con otros
- **Una lista de texto** para tareas del día
- **Nada más**

La productividad no está en la herramienta — está en la **claridad** de saber qué hacer y la **disciplina** de hacerlo.
