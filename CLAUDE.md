# www.rodhappi.com — Personal Blog

## Project Overview

A personal blog (`www.rodhappi.com`) — a static site covering AI, tech, and topics relevant to Latin America. Built with the AI-Assisted Development Framework as development tooling (see `.claude/` for workflow skills).

- **Stack:** Vanilla HTML/CSS/JS, zero dependencies
- **Build:** Custom Node.js script (`node build.js`) — no `npm install` needed
- **Language:** Spanish (`lang="es"`)
- **Design:** Minimalist editorial style (Merriweather serif, JetBrains Mono code)

## Architecture

### Site Structure

```
content/           ← Source Markdown posts (edit here)
templates/         ← HTML shells with placeholders (index.html, post.html)
css/style.css      ← Complete design system (typography, layout, animations)
js/nav.js          ← Navigation toggle (slide-out menu)
media/             ← SVGs, images, diagrams
build.js           ← Build pipeline (Markdown parser + template engine)
posts/             ← GENERATED — never edit directly
index.html         ← GENERATED — never edit directly
```

### Key Rule: Source vs Generated

- **Edit:** `content/*.md`, `templates/*.html`, `css/style.css`, `js/nav.js`, `build.js`
- **Never edit:** `posts/*.html`, root `index.html` — these are regenerated by `build.js`

## Build & Deploy

```bash
node build.js          # Parse content/*.md → generate posts/*.html + index.html
```

Output: `✓ N post(s) built successfully.`

Deploy to any static host (GitHub Pages, Netlify, Vercel) — no build config needed.

## Content Conventions

### Post Format

```yaml
---
title: "Post Title"
date: 2026-02-26
description: "Meta description for listings and SEO"
featured: true
---

Markdown content here...
```

### Supported Markdown

Standard: headings (`##`, `###`), bold, italic, links, images, code blocks, lists, blockquotes, horizontal rules. Custom: `@video(url)` for embedded video (YouTube/Vimeo/local).

### Publishing Logic

- Most recent `featured: true` post → "Escritos" (featured section)
- All others → "Archivo" (archive), sorted newest first
- No featured post? Most recent auto-promotes

### File Naming

- Content: `content/my-post-slug.md` → generates `posts/my-post-slug.html`
- All files: kebab-case, lowercase
